
* Reference: 《致命公司》，《群星Stellaris》，《三伏》
* Game Abstract: 你是一个alien,驾驶着宇宙飞船，来到2100年的地球，此时地球人已经因为环境和科技因素离开地球，地球本身环境开始恶劣。
	由于科技树点歪，地球的许多东西对你都很新奇且危险。
	你的目的是在地球上用偏门科技过原始人的生活。
	地球上的道具能帮助你在"孤岛"生存,正确的决策决定你能否生存到同伴救援的日期。
	作为一个alien，你是一个数字化生物，能够通过消耗能源反复初始化，但是你却需要通过摄入最基本的蛋白质来获取能量。地球的不良天气对你来说
	都是致命性毒药。
* Game Type: 3D,探索，策略，第一人称
* Game Core: 人性中的善美，恐怖，科幻
* Game Flow: Base(基地，任务，仓库)->大场景->物品道具收集->死亡事件 -- 重复尝试-> 结束单局基本任务，收集 ---多局游玩结束，结局。
  e.g. 找火柴-生火-> 寻找柴 -> 驱赶巨兽/制作食物,建造障碍/房子抵御不良天气-> 学习地球知识/读书 ->积攒能量值，回到基地，分配能量，
  能量耗尽视为死亡。用外星人的科技，"小孩"的认知，探索荒废了一段时间的地球。
* Game Feature:
  1. 基地系统: 控制台UI,控制台管理物品+消息通知，基地场景
  2. 场景设计: 天气系统，基本场景模型，障碍物及生成系统。
  3. 道具系统：基础消耗类：生火道具，书,肉; 决策剧情类: 能量,知识,情绪，机能；
